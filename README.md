# MV Custom Theme

A modern WordPress theme that delivers a polished landing page, testimonial highlights, and a blog preview in a single lightweight package.

## Features

- Hero section with call-to-action buttons
- Responsive grid-based feature highlights and testimonials
- Recent posts preview with automatic excerpts
- Custom logo support, navigation menus, and footer widget area
- Google Fonts typography and cohesive color system

## Getting started

1. Copy the theme folder into your WordPress installation's `wp-content/themes/` directory.
2. Log in to the WordPress admin dashboard and activate **MV Custom Theme** from the Appearance → Themes screen.
3. Assign menus to the **Primary Menu** and **Footer Menu** locations under Appearance → Menus.
4. Add widgets to the **Footer Widgets** area if desired.
5. Set a static front page that uses the built-in front page template, or let the theme display your latest posts by default.

## Development

- All styles are stored in `style.css`; update the CSS variables at the top to quickly tweak the color palette or typography.
- Scripts and fonts are enqueued via `functions.php`.
- Template files follow standard WordPress conventions (`front-page.php`, `index.php`, `single.php`, `page.php`).

Enjoy building with MV Custom Theme!
